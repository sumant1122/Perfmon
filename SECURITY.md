# Security Policy

## Supported Versions

Currently, we support the latest version of Perfmon.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |

## Reporting a Vulnerability

If you've found a security vulnerability in Perfmon, please do not open a public issue. Instead, report it privately.

To report a vulnerability, please open a GitHub issue with the "Security" label (if available) or contact the maintainer directly if an email address is provided in the repository profile.

When reporting, please include:
- A description of the vulnerability.
- Steps to reproduce it.
- Possible impact.

We will acknowledge your report within 48 hours and provide a timeline for a fix.
